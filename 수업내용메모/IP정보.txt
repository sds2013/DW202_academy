DB접속하려면

1. IP
내가 DB에 접속하려면 
DB가 설치된 콤푸터에 IP를 알아야 함
 192.168.0.32 == localhost

2. PORT(방)
 3306
 192.168.0.32:3306 입력해야 
192.168.0.32 주소를 가진 컴푸터 3306 방에 들어갔다 

3. (option) database 이름
DB =>데이터베이스 -> 테이블 -> 데이터
이클립스 => 프로젝트 -> 패키지 -> 클래스

IP가 두 종류가 있음
 
1. private IP
2. public IP
